TS_TextComposite
================
## 概要
canvasに文字合成をおこなうテスト

## 検証内容
- canvasにinputから取得した文字列を合成、画像として出力する
- retinaで画像が引き延ばされてしまう対策
- 縦書きの検証(プレビューエリアとcanvasの差をプレビュー側で調整)
- 縦書きの検証(プレビューエリアとcanvasの差をcanvas側で調整)..いつかやりたい
- スライダー・ボタンサイズ変更を追加したもの
